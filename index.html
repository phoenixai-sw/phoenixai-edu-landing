About Phoenix AI

AI 기술 전문 기업

Course 1

Business Basic Course

4주 완성 멀티미디어 AI 마스터 과정

📚 커리큘럼

Week 1: Text 생성형 AI
  • 도구: ChatGPT, Gemini, Claude, GPTs
  • 주요 활용: 문서 요약·이메일·사업계획서·PPT 초안 작성
  • 자동화 실습: GPTs + Zapier 연동 뉴스 분류·발행 시스템 구축
Week 2: Image 생성형 AI
  • 도구: DALL·E 3, Midjourney, GPTs
  • 주요 활용: 화보·가상 모델·굿즈 디자인·브랜딩 비주얼 제작
  • 자동화 실습: 외부 API 연동 이미지·텍스트 수집 프로세스 구현
Week 3: Video 생성형 AI
  • 도구: Kling AI, GPTs, YouTube API
  • 주요 활용: 예고편·광고 영상·특수효과·패러디 제작
  • 자동화 실습: YouTube API 기반 영상 요약·분류 시스템 구축
Week 4: Audio 생성형 AI
  • 도구: Suno AI, Make.com
  • 주요 활용: AI 성우·CF 음악·주제곡·팟캐스트 제작
  • 자동화 실습: Make.com 활용 뉴스 요약 → 자동 배송 서비스 완성
